# Arroyo Vista IT — Central Hub

An internal IT support portal for **Arroyo Vista Family Health Center**. Staff can browse the
NextGen knowledge base, find IT team contacts, and submit support tickets that route to
`itstaff@arroyovista.org`.

## What's inside
- `index.html` — the main page (open this to view the site)
- `app.js` — application logic (boot animation, ticket form, knowledge base, etc.)
- `data.js` — editable content: team members, FAQs, NextGen knowledge base, ticket categories
- `assets/` — logos
- `fonts/` — bundled fonts

## Running it
It's a fully static site — no server or build step needed. Just open `index.html` in a browser,
or host it for free with **GitHub Pages**.

### Publishing with GitHub Pages
1. Push these files to a GitHub repository.
2. In the repo, go to **Settings → Pages**.
3. Under "Build and deployment", set **Source** to *Deploy from a branch*.
4. Choose the `main` branch and the `/ (root)` folder, then **Save**.
5. Wait ~1 minute — your live URL appears at the top of the Pages settings.

## Editing content
Most text lives in `data.js` — open it in any editor to update team members, phone extensions,
FAQs, or the NextGen knowledge base entries. The IT team and general FAQ sections can also be
edited in-app through the admin panel.
